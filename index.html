Combined Vietnamese Sources for Paperback

Add this extension to Paperback to read manga from TruyenGG, MimiHentai, and NHentai.

Installation

To add this source to Paperback:

  1. Open Paperback
  2. Go to Settings > Sources > Add a new source
  3. Enter this URL: https://Volcano1701.github.io/vietnamese-sources-extension
  4. Click "Add"
  5. Install the extension
